Project Cloner

Clone QGIS projects (layers, styles & data) into a portable ZIP — ready to move between machines or share.

About

Project Cloner packages the current QGIS project file and associated data (shapefiles, GeoTIFFs, style files, etc.) into a single ZIP for backup or transfer.

Key Features

  • Auto-detects project file and suggests a timestamped ZIP filename
  • Option to include or exclude layer data files
  • Preserves layer styles and project settings
  • Simple UI with progress and status messages

Install

  1. Download the latest ZIP from Releases or use the QGIS Plugin Manager.
  2. Unzip to your QGIS plugins folder (usually ~/.local/share/QGIS/QGIS3/profiles/default/python/plugins/ on Linux).
  3. Restart QGIS and enable Project Cloner from the Plugin Manager.

Usage

Open the plugin from the Plugins toolbar. Choose where to save the cloned ZIP, select whether to include data files, and click Clone Project. If your public IP resolves to Madagascar, you'll see a Malagasy greeting.
